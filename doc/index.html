Module autopickup

A system which allows to automatically pickup items which are on the ground.

Functions

activate () Activates the system, if it has not been disabled by setting "autopickup_activate" to false in the configuration.
has_just_been_dropped_by (entity, player) Checks if the given entity has just been dropped by the given player within the configured time.
move_towards (object, target_pos) Moves the given object towards the given location with the given velocity.
pickup_items (player) Processes all items in the vicinity of the given player.
pickup_items_all () Processes all players and picks up the items.
play_sound (position, player) Plays the configured sound at the given position.

Fields

activate If the system should be automatically activated.
active If the system is active/has been activated.
attraction_acceleration The acceleration that is used when the item enters the attraction radius, defaults to "2, 0, 2".
attraction_radius The radius within which items are moved towards the player, defaults to 1.5.
dropped_min_age The minimum age (in seconds) before dropped items can be picked up automatically again by the same player, defaults to 4.
interval The interval (in seconds) in which the system is running, defaults to 0.1.
pickup_radius The radius within which items are picked up, defaults to 0.9.
sound_enabled If the sound should be played, defaults to true.
sound_gain The gain of the sound played, defaults to 0.5.
sound_global If the sound be played globally, defaults to true.
sound_hear_distance If the global playing is set, this defines the hearing distance, defaults to 8.
sound_name The name of the sound to play, defaults to "auto_pickup".
